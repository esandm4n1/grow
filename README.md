# GROW 🏀 · Fukui Kings Edition

**「鍛える」のではなく「成長を邪魔せずに伸ばす」**

福井県バスケットボールクラブ U12 Fukui Kings (フクイキングス) 所属選手のための、成長期に配慮した練習・記録アプリ。

## 新しい機能 (v4)

- **Fukui Kings テーマ**: 黒 × Kings レッドのダークデザイン
- **公式ロゴ組込み**: フクイラプトルのロゴを随所に配置
- **選手プロフィール**: 顔写真アップロード & 背番号入力
- **盾型背番号バッジ**: 王冠つきのユニフォーム風デザイン
- **完了アニメ強化**: ロゴがドーンと出現、パーティクル、集中線
- **"NEVER GIVE UP" モットー**: ネオン点滅する映画タイトル風バッジ

## 機能

1. **ホーム** - 選手プロフィール、今日の予定、連続記録
2. **今日の練習** - 自動生成メニュー (3〜5種目 / 10〜20分)
3. **日次記録** - 睡眠・疲労・痛み・食事・ジャンプ・身長
4. **成長グラフ** - 身長 / ジャンプ力 / 睡眠の推移
5. **習慣チェック** - 連続記録バッジ、30日カレンダー
6. **設定** (新規) - 名前、背番号、顔写真、試合日、バックアップ

## メニュー生成ロジック

| 状態 | 対応 |
|---|---|
| 睡眠<5h or 疲労MAX | 完全休養メニュー |
| 膝/足首の痛み≥2 | ジャンプ全禁止 |
| 疲労≥4 or 睡眠<7h | 低負荷モード |
| 2日連続ジャンプ | 3日目は回復日 |
| 試合前日 | 軽い刺激のみ |
| 試合当日 | ストレッチのみ |

## 使う技術

- React 18 + Vite
- Tailwind CSS (ダークテーマ + Kings カラーパレット)
- Recharts (グラフ)
- localStorage (データ保存、写真含む)
- Google Fonts (Oswald / Bebas Neue / Staatliches / Noto Sans JP)

## ローカル実行

```bash
npm install
npm run dev        # 開発サーバー
npm run build      # 本番ビルド → dist/
npm run preview    # ビルド結果をプレビュー
```

## GitHub Pages へデプロイ

```bash
npm run build
# dist/ フォルダの中身を gh-pages ブランチにデプロイ、
# または docs/ にコピーしてリポジトリ設定から公開
```

## 注意事項

- 小学生の身体は大人と違います。痛みがあるときは無理させないでください
- 身長を「伸ばす」と断言する医学的根拠のある方法はありません
- このアプリは「成長を邪魔しない習慣づくり」を目的としています
- 気になる痛み・違和感がある場合は専門医 (整形外科・スポーツドクター) にご相談ください
- Fukui Kings ロゴはチーム内配布専用として使用。他チーム・外部公開時はチーム代表に確認してください
